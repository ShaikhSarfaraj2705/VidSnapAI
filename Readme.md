

# 🎬 VidSnapAI – AI Reel Generator

VidSnapAI is a Python-based web application built using Flask that allows users to create AI-generated reels from uploaded images and text. It automates media processing and can generate audio, visuals, and structured reels for social media content creation.

---

## 🚀 Features

- 📸 Upload images for reel creation
- 🎙️ AI text-to-speech generation (audio narration)
- 🎞️ Automatic reel generation workflow
- 🧠 Simple and clean Flask backend
- 📂 Organized file storage system
- 🎨 User-friendly web interface

---

## 🛠️ Tech Stack

- Python 🐍
- Flask 🌐
- HTML, CSS, JavaScript 🎨
- ElevenLabs API (for AI voice generation)
- FFmpeg (for video processing)
- UUID for unique file handling
